How it fits together
The full picture

Each tool is independent, but they're designed to work together. LapTime informs the hardware decision. Cruise Control manages the network once it's deployed. AI Hub is the daily operations surface. Runway keeps quota burn visible. AIKB is the memory layer underneath all of it.
